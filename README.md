## Description
The game was created as a training exercise to practice with DOM manipulation and implementing functionality using JavaScript. 

## How to play
* Click Start Game to start.
* The cards dealt will be displayed under cards.
* Click hit if you'd like to be dealt another card.
* Click Play Again to reset the game.
